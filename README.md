# Heart Disease Prediction

## Project Overview
This project predicts whether a patient has heart disease using machine learning algorithms.

## Dataset
The dataset contains patient health information such as:
- Age
- Cholesterol
- Blood Pressure
- Chest Pain Type
- Resting ECG
- Maximum Heart Rate

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest (Best Model)
- XGBoost
- SVM(Support Vector Machine)
## Best Model: Random Forest
- Accuracy: 89%
- Recall: 94%
- Selected as best model based on highest recall for healthcare prediction

## Why Recall is Important
In healthcare, recall is more important because we must correctly identify patients who actually have heart disease. Random Forest achieved highest recall.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Conclusion
Random Forest model performed best with high accuracy and recall, making it suitable for early heart disease prediction.
